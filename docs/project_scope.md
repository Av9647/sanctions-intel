# Project Scope - SanctionsIntel

## What this project does
- Ingests public global sanctions datasets for research purposes
- Normalizes regulatory data into analytical lakehouse tables
- Extracts narrative text for semantic embedding and similarity analysis
- Demonstrates retrieval-augmented analysis workflows grounded in regulatory disclosures to support KYC and EDD analysis
- Emphasizes transparency, reproducibility, and explainability

## What this project does not do
- Perform sanctions screening or compliance decisions
- Replace rule-based AML or KYC systems
- Score, rank, or label entities as high-risk
- Provide real-time monitoring or alerting
- Operate as a production-grade system

## Design principles
- Deterministic transformations over automation
- Explainability over model complexity
- Research-grade implementation over performance optimization
